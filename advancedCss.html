CSS Can Do Amazing Things!

The web has come a long way since its humble beginnings. We now have a massive array of tools at our finger tips for designing and making any website look exactly how we imagine it! CSS (Cascading Style Sheets) is one of those innovative tools we have. The newest version, CSS3, allows for many special effects to be executed using CSS and HTML alone. Animations that once required a knowledge of Javascript and JQuery can now be done in raw CSS. Take the "Hover On Me!" button down to the left. Using only CSS, when the div is hovered over with the mouse it will change it's background color, text color, and location on the screen over the course of 1 second! Go ahead, try it!
