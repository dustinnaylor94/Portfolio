Learning React at Work

In my internship with the Church of Jesus Christ of Latter-Day Saints I have been learning React.
React is a Javascript framework that allows for the creation of reactive, reusable components that actively update whenever a change is made. Learning React has been a big, but exciting, new challenge for me. I enjoy learning how this framework communicates with a database to keep page elements live and up to date
